# LynxHub Card Module - Production Build

This branch contains the compiled production build of the card module.

## Compiled Structure

The compiled assets must be structured as follows:

```
scripts/
├── main.mjs      # Compiled main process script
└── renderer.mjs  # Compiled renderer process script
```

These assets are generated by running:
```bash
npm run build:module
```
in the host application repository, and should be published to this branch for production distribution.
